struct Prediction

HomeView is a View struct that displays the home page of the application. This homepage shows the user its receipts, the folders, the title bar (doubling as a search bar).

  • Called by ContentView.
  • pointPrediction(text:)

    pointPrediction is the parent function that manages the processing of the receipt text to predict it’s folder assignment. It first uses predictedCategories to to get an idea of the various categories it may be associated with, then checks each category’s number of matches and assigns a best prediction based on the matches.

    Declaration

    Swift

    static func pointPrediction(text: String) -> String

    Parameters

    text

    the body of text (extracted from a receipt) to be processed and have the category predicted from.

    Return Value

    A string holding the predicted folder title the receipt should be assigned to.

  • predictedCategories(text:)

    predictedCategories is a function that takes a text input, and compares it against the categoryKeywords array to assign a point to each category depending on if it matches with it’s keywords.

    Declaration

    Swift

    static func predictedCategories(text: String) -> [(String, Int)]

    Parameters

    text

    the body of text (extracted from a receipt) to be processed to form the number of folder predictions.

    Return Value

    [(String, Int)] of tuples, which contain the Categories title, along with the number of matches the text had with its keywords.

  • matchString(keywords:input:)

    matchString is a function that counts how many matches a word . It functions by looping over each word in the categories keywords, incrementing a counter for each matching word. The count is returns at the end of the loop.

    Declaration

    Parameters

    Return Value

    [(String, Int)] of tuples, which contain the Category title, along with the number of matches the input text had with its keywords.

  • categoryKeywords

    categoryKeywords is an array of tuples that are used as the basis for the folder prediction process. It is made up of a title of the category, along with its keywords associated with the category.

    • Tuple format
      • (title: “Categories Title”, keywords:[“keywords”, “associated”, “with”, “the”, “category”])
