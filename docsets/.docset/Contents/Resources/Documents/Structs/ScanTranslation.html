ScanTranslation

struct ScanTranslation

ScanTranslation is a struct that is used to take an array of scanned images, extract their information, and stitch them into the RecognizedContent object for the ScanView structs to utilize.

  • Called by GalleryScannerView, and CameraScannerView.
  • scannedImages

    scannedImages is an array of UI Images that contains the images of the receipts the user has scanned.

    recognizedContent is an object that holds arrays of ReceiptItems, which contain the text and image of a processed receipt.

  • recognizeText()

    recognizeText is a function that is used to recognize text in the scannedImages parameter of the ScanTranslation struct. It loops through all images in scannedImages, creates a ReceiptItem, and sets the image to the ReceiptItem before calling getTextRecognitionRequest to extract and set the images text.
